Training Room 5—Tables

 

Time to Practise

 

 

Now that you have an understanding of how to create tables, the practice activities below will help you apply your knowledge.

 

Practice 1

 

In this practice, you will be asked to create the following table by setting and using Left, Center, and Right tabs with provided content. You may go back to the demonstrations and check steps if you need assistance.

  1. Open a new, blank Microsoft Word® 2007 document.

  2. Save the document as “Table 1” in your INF1030: WP1: Tables folder.

  3. Create and format the following table using tabs. 


  1. Make sure your line spacing is set to single (Home tab, Paragraph group in the ribbon).

  2. Set the following tabs on your ruler or using the Tab dialog box: Left tab .5”; Center tab 3”; Right tab 6”.

  3. Type the title: “MICROSOFT OFFICE” — bold, centre, uppercase.

  4. Hit the Enter key twice to leave one blank line between the title and the subtitle.

  5. Type the subtitle: “Information Processing” — centre, Capitalize Each Word.

  6. Hit the Tab key to move the cursor to the first tab setting.

  7. Enter the column headings: “Module,” “Program,” and “Application,” using the Tab key to move to each column.  

  8. Enter the body text for the table using the Tab key to move to each column. Use normal type for the body (not bold).

  9. Centre the table vertically on the page using the Page Layout group: click the Page Setup dialog box and select the Layout tab.

  10. Use the Spelling and Grammar check and proofread the table to ensure the information is correct.

  11. Print Preview the letter to see it in its final form.

  12. Save your document.
Practice 2

 

In this practice, you will be asked to create a table by using the Table function with provided content.

  1. Open a new, blank Microsoft Word 2007 document.

  2. Save the document as “Table 2” in your INF1030: WP1: Tables folder.

  3. Click the Insert Tab, Table button.

  4. Create a table with three columns and two rows.  Select Autofit to Contents.

Microsoft product screen shot(s) reprinted with permission from Microsoft Corporation.

 


  1. Practise inserting an additional four rows to the table after it has been created. (You might want to review the demonstrations if you have forgotten how to do this.)

  2. Enter the following information into your table.

 

 


  1. Select all three columns in the first row of the table and, in the Table Tools tab, select the Layout tab, then select Merge Cells. The three cells merge into one cell.

 

Microsoft product screen shot(s) reprinted with permission from Microsoft Corporation.



  1. In the Table Tools Layout tab, adjust the alignment of your text in the cells to center.

  2. Bold the title and column headings in your table.

  3. Use the Table Tools, Design tab to apply Shading to the title row and a complimentary shade to the column headings row.

  4. Centre the table horizontally on the page using the center button in the Paragraph group or Ctrl + E.

  5. Vertical-centre the table using the Page Layout tab — look for the Center button in the Alignment group. 

  6. Spell check and proofread the table to ensure the information is correct.

  7. Make any formatting adjustments you feel are necessary to enhance your table.

  8. Print Preview the table to see it in its final form.

  9. Save the document.
Practice 3

 

In this practice, you will create an attractively formatted contact list for a minimum of five friends using a table. You can create your table by setting the tabs or you can use the Table command — the choice is yours. Your address book table must meet the following criteria:

  1. Include a table title and appropriate column headings. You must include at least four columns of information, but you may choose to include more. Some suggestions for column headings are phone numbers, birthdays, email addresses, street addresses, or any other important, appropriate information.

  2. Format to your table to make it visually appealing.

  3. Horizontal- and vertical-centre the table on the page.

  4. Save the document as Table 3 in your INF1030: WP1: Tables folder.
