Training Room 3: Functions and Formulas in Spreadsheets

 

© Andreas G. Karelias/shutterstock

Each training room you complete adds to the skill set you are developing in working with spreadsheets. In Training Room 3 you will advance your knowledge of formulas by using more functions within the AutoSum function, by making use of date and time formulas, and by applying logic and financial formulas.

 

In this training room you will

  • study common functions
  • work with minimum and maximum formulas
  • calculate averages
  • use functions to find minimum, maximum, and average
  • summarize data using subtotals
  • create formulas for dates and time
  • use functions to count both text and numbers
  • use functions to calculate monthly payments, interest rates, and present value
  • use conditional logic in a formula (e.g., IF statements)
  • format text using text functions
  • look up data using formulas
  • protect cells
How to Do It

 

To successfully complete this training room, do the following:

  • Read and watch instructional videos of topics covered in the training room.
  • Use help and online references should you require additional instruction or software assistance.
  • Complete the Time to Practise 3 exercise.

You are now ready, once again, to start working in Microsoft Excel 2007. Open Training Room 3: Functions and Formulas in Spreadsheets, and follow the instructions, starting with common functions. Save your Training Room 3 in your INF2080 SS2 folder.
